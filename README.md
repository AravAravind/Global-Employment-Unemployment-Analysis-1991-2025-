# 🌍 Global Employment & Unemployment Analysis (1991–2025)

This project analyzes global employment and unemployment trends using country-level data.

## 🔍 Key Features

* Global employment vs unemployment trend analysis
* Country-wise unemployment comparison
* Crisis impact analysis (2008 Financial Crisis & COVID-19)
* India vs World unemployment comparison
* Job market risk heatmap
* Recovery speed model (post-crisis resilience)
* COVID recovery analysis across countries
* Future unemployment forecasting

## 📊 Insights

* Unemployment spikes align with major global crises
* COVID had a stronger and more immediate impact than 2008
* Countries show different recovery speeds after shocks
* India demonstrates unique recovery patterns compared to global trends
* Some countries recover quickly, while others remain structurally weak

## 🛠 Tech Stack

* Python
* Pandas
* Matplotlib / Seaborn
* Scikit-learn

## 💡 Conclusion

This project highlights how global labor markets respond to economic shocks and measures how quickly countries recover, providing insights into economic resilience.
